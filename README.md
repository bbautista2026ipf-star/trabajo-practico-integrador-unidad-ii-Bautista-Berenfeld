# Trabajo Práctico: Sitio Web Institucional IPF

**Materia:** [Taller de Lenguajes de Programacion I]  
**Año:** 2026  
**Alumno:** Bautista Berenfeld

## Descripción del Proyecto

Diseño, maquetación y desarrollo front-end del sitio web institucional para el Instituto Politécnico Formosa (IPF). El proyecto consiste en la creación de un sistema de páginas interconectadas (Inicio, Institucional, Carreras y Contacto) implementando interfaces responsivas, navegación estructurada y un diseño coherente para presentar la oferta académica de la Tecnicatura Superior en Desarrollo de Software Multiplataforma y demás carreras de la institución.

## 🛠️ Tecnologías Utilizadas

- **HTML5:** Estructura y semántica de las páginas web.
- **CSS3:** Hoja de estilos en cascada para el diseño visual personalizado.
- **Bootstrap 5.3:** Framework frontend para el diseño responsivo, sistema de grillas y componentes interactivos (carrusel, menús desplegables, tarjetas, acordeón).
- **Git / GitHub:** Control de versiones y alojamiento del repositorio.
